Annotator-Segmentator

Utworzony w ramach mojej pracy w projekcie Clarin. Stanowi dodatek do programu "Elan". Głównym zadaniem jest częściowa automatyzacja pracy związanej z anotacją nagrań rozmów nagrywanych w ramach korpusu Clarin. Program oferuje prosty interfejs graficzny, głównym zadaniem jest analiza nagrania wav, celem wykrycia mowy, (wykrycie kiedy jest mowa, a nie rozpoznanie tej mowy), a następnie przygotowanie pliku .eaf który to może być edytowany w pogramie Elan. Program skraca czas anotacji nagrania o około 40%. Produkt został wydany w wersjach portale na systemy Windows i Linux.
  • Python
  • QT

Lady Isabell bot v3

Dziś jest to sieć botów na komunikatorze Telegram, których celem jest pomoc organizatorom w przeprowadzeniu eventów. Oferuje możliwość utworzenie i dowolnej customizacji do 3 botów. Głównym celem jest udzielanie odpowiedzi na powtarzające się pytania, oprócz tego umożliwia prowadzenie zapisów, weryfikacji uczestników, autonomicznej moderacji grup dyskusyjnych i wiele innych. Początkowo rozwijany jako bot Q&A na Telegramie w oparciu o Php i MySQL, z czasem co raz bardziej rozwijany. Wersja v2 stanowiła refaktoryzację całego kodu źródłowego na język Python i PosgreSQL. Wersja v3 umożliwiła klonowanie instancji botów, oraz umożliwiła pełną customizację z interfejsu Telegrama.
  • Python
  • FastAPI
  • PostreSQL
